## 🎯 Your Next Steps (5 Minutes)

### **1. Open VS Code Terminal**

In VS Code, press: `Ctrl + ~`

You should see a PowerShell terminal at the bottom.

### **2. Navigate to Your Project**

Copy and paste this command:

```powershell
cd C:\smart_chatbot
```

Then press Enter.

### **3. Check All Files Are There**

```powershell
dir
```

You should see:
- app.py
- database.py
- requirements.txt
- static/ (folder)
- templates/ (folder)
- README.md
- QUICKSTART.txt
- And more...

### **4. Create Virtual Environment**

Copy and paste:

```powershell
python -m venv venv
```

Press Enter and **wait 30-60 seconds**.

### **5. Activate Virtual Environment**

Copy and paste:

```powershell
.\venv\Scripts\activate
```

You should now see:
```
(venv) C:\smart_chatbot>
```

### **6. Install Dependencies**

Copy and paste:

```powershell
pip install -r requirements.txt
```

**Wait 1-2 minutes** for it to finish.

### **7. Start the Flask App**

Copy and paste:

```powershell
python app.py
```

You should see:
```
============================================================
🤖 Smart FAQ Chatbot System Starting...
============================================================
📍 Open your browser and go to: http://localhost:5000
============================================================
```

### **8. Open Your Browser**

Open Google Chrome, Firefox, or Edge and go to:

```
http://localhost:5000
```

**🎉 Your chatbot is live!**

## 🐛 Something Not Working?

### Error: "python not found"
→ Install Python from https://www.python.org/downloads

### Can't activate venv
→ You're on Command Prompt, not PowerShell. Use: `venv\Scripts\activate.bat`

### Port 5000 in use
→ Close other Flask apps or restart your computer

### Module 'flask' not found
→ Make sure `(venv)` shows in your terminal before running pip

### Database error
→ Delete `database.db` and restart Flask

**Full troubleshooting:** See [README.md](README.md)
